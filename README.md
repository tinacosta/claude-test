# Soprano Ornamentation Exercises

An advanced vocal training web application designed for soprano singers to practice ornamentation techniques from multiple musical traditions.

## Features

### 10 Diverse Exercises

The app includes exercises from various musical traditions:

1. **Baroque Trill - Italian Style**: Classic upper-neighbor trills with acceleration
2. **Bel Canto Portamento**: Smooth gliding between notes with connected tone
3. **Coloratura Run**: Rapid ascending and descending scale passages
4. **Messa di Voce with Turn**: Dynamic control with ornamental turns
5. **Appoggiatura Sequence**: Leaning notes resolving downward
6. **Jazz Scoop and Fall**: Contemporary slides with vibrato
7. **French Ornamental Grace Notes**: Delicate baroque-style grace notes
8. **German Lieder Mordent**: Expressive alternation with lower neighbors
9. **Spanish Melisma**: Flowing melismatic passages with passion
10. **Staccato Pearls**: Brilliant coloratura staccato technique

### Vocal Range

- **Starting Note**: Middle C (C4 - 261.63 Hz)
- **Highest Note**: D6 (1174.66 Hz)
- **Lowest Note**: A3 (220 Hz)
- **Progression**: Ascends from C4 to D6, then descends to A3

### Smart Practice Features

- **Automatic Progression**: Hands-off operation through the entire range
- **Breathing Cues**: Automatic breathing reminders at appropriate intervals
- **Vowel Guidance**: Suggested vowels/phrases for each exercise
- **Pattern Display**: Real-time visualization of the current ornamentation pattern
- **Progress Tracking**: Visual progress bar showing your position in the exercise

### Playback Controls

- **Start/Resume**: Begin or continue the exercise
- **Pause**: Temporarily stop the exercise
- **Restart**: Begin the current exercise from the start
- **Tempo Control**: Adjust tempo from 40-120 BPM

### Audio Synthesis

- Uses Web Audio API for pure sine wave generation
- Dynamic envelopes for natural sound
- Special effects including:
  - Vibrato on sustained notes
  - Glissando for portamento passages
  - Crescendo/diminuendo for messa di voce
  - Staccato articulation for pearl exercises

## How to Use

1. **Open the App**: Simply open `index.html` in any modern web browser
2. **Select an Exercise**: Choose from the dropdown menu
3. **Adjust Tempo**: Use the slider to set your preferred tempo (start slower!)
4. **Read the Description**: Each exercise includes guidance on technique and tradition
5. **Press Start**: The exercise will run automatically through the entire range
6. **Follow Cues**: Watch for breathing cues and vowel suggestions
7. **Pause if Needed**: Use pause/restart controls as necessary

## Technical Details

- **Technology**: Pure HTML5, CSS3, and JavaScript (no dependencies)
- **Audio**: Web Audio API with oscillator-based synthesis
- **Compatibility**: Works in all modern browsers (Chrome, Firefox, Safari, Edge)
- **Responsive**: Adapts to different screen sizes

## Practice Tips

### For Advanced Vocalists

- Start with slower tempos and gradually increase speed
- Focus on evenness and clarity in ornamentation
- Use the breathing cues to develop efficient breath management
- Experiment with different vowels for each exercise
- Record yourself to monitor progress
- Practice each exercise type before attempting all ranges

### Warm-Up Suggestions

1. Begin with **Bel Canto Portamento** for smooth connection
2. Progress to **Baroque Trill** for flexibility
3. Challenge yourself with **Coloratura Run** and **Staccato Pearls**
4. Cool down with **Messa di Voce** for control

### Musical Traditions Represented

- **Italian**: Baroque trills, bel canto techniques, coloratura
- **German**: Lieder mordents, expressive ornamentation
- **French**: Baroque grace notes, delicate articulation
- **Spanish**: Melismatic passages, folk-inspired ornamentation
- **Jazz/Contemporary**: Scoops, falls, modern vocal techniques
